    Class BaseSpecCommand
 
Base class for SpecCommand objects

__call__(self, func=False, *args)
Execute the command with the arguments
 
Build a command string to be executed by Spec, and call
self.executeCommand on it. The given arguments are
concatenated with the command name (self.command).
 
Keyword arguments:
func -- if True, indicates that self.command is a macro function.
Otherwise, indicates that self.command is a macro. This changes the way
the final command string is built. Defaults to False.
